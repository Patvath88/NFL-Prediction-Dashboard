
# NFL Prop Predictor Pro+

An enhanced deployable Streamlit app that predicts NFL player props using live odds and Sportsdata.io stats.
Includes PrizePicks-style slip builder, fuzzy player mapping, and expanded market coverage.

## Run Locally
```bash
pip install -r requirements.txt
export ODDS_API_KEY=your_theoddsapi_key
export SPORTSDATA_API_KEY=your_sportsdata_key
streamlit run app.py
```
